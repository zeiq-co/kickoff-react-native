# kickoff-react-native

## Requirements

- NodeJs
- Npm / Yarn
- Expo

## Setup

```
yarn install
```

## To run

iOS

```
yarn ios
```

Android

```
yarn android
```

Web

```
yarn web
```

## Build

```
expo build:android
expo build:android -t app-bundle
or
expo build:android -t apk

expo build:ios
```

## Stacks

- [tailwindcss-react-native](https://tailwindcss-react-native.vercel.app/)
- [vector-icons](https://icons.expo.fyi/)
